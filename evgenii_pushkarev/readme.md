# RetroArcade - Магазин старых игровых автоматов

## Описание проекта
Веб-приложение для продажи и сравнения старых игровых автоматов. Проект включает систему регистрации пользователей, добавление товаров, фильтрацию и сравнение товаров.

## Функциональность
- Каталог товаров с фильтрацией
- Регистрация и авторизация пользователей
- Личный кабинет с возможностью редактирования данных
- Добавление новых товаров на сайт
- Сравнение товаров
- Адаптивный дизайн
